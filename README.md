## Customization

### Change Profile Information
- Edit `index.html` section tags (About, Education, Publications, Research)
- Replace profile image at `assets/images/profile/`
- Update logos in `assets/images/logos/`

### Modify Styling
- Edit CSS variables in `:root` block
- Change colors, spacing, fonts in `desktop.css` or `mobile.css`
- Changes apply site-wide instantly

### Update Links
- Publication URLs: Find in Publications section
- Social links: Update in About section
- Research links: Modify in Research section
