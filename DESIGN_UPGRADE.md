## 🎯 设计原则

### 1. **一致性**
- 所有页面使用相同的配色方案
- 统一的组件设计（卡片、按钮、标签）
- 一致的间距和排版

### 2. **现代性**
- 阴影和深度感
- 平滑的动画和过渡
- 现代的色彩搭配（蓝色 + 金色）

### 3. **可用性**
- 清晰的视觉层级
- 充足的对比度
- 友好的触摸目标

### 4. **性能**
- 使用 CSS 变量减少代码重复
- 高效的动画（使用 transform 和 opacity）
- 最小化的重排和重绘
